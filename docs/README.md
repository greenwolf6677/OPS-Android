# OPS - Orders Processing System

![OPS Logo](assets/icon/logo.png)

**نظام معالجة الطلبات المتكامل لإدارة المبيعات والمشتريات والمخزون**

---

## 📱 عن النظام

OPS هو نظام متكامل لإدارة المطاعم والمتاجر والسوبرماركت، تم تطويره بلغة Python باستخدام إطار Kivy ليعمل على منصات متعددة (Android، Windows، Linux). يوفر النظام واجهة سهلة الاستخدام باللغة العربية مع دعم كامل للغة العربية.

### المميزات الرئيسية

- ✅ **إدارة المبيعات** - نظام متكامل للمبيعات مع سلة مشتريات
- ✅ **إدارة المنتجات** - إضافة وتعديل وحذف المنتجات مع إمكانية إضافة الصور
- ✅ **إدارة الزبائن** - متابعة أرصدة الزبائن والمدفوعات
- ✅ **إدارة الموردين** - متابعة المشتريات والمدفوعات للموردين
- ✅ **إدارة المشتريات** - تسجيل فواتير الشراء وتحديث المخزون تلقائياً
- ✅ **نظام المرتجعات** - إدارة مرتجعات المبيعات وتحديث الأرصدة
- ✅ **التقارير المتعددة** - تقارير المبيعات، الأرباح، الديون، حركة المخزون
- ✅ **إدارة المستخدمين** - صلاحيات متعددة (مدير/موظف)
- ✅ **نظام الترخيص** - نسخة تجريبية 14 يوم مع إمكانية التفعيل
- ✅ **طباعة الفواتير PDF** - دعم كامل للغة العربية في الفواتير

## 📋 المتطلبات

### للتشغيل المباشر (Windows/Linux)
- Python 3.8+
- Kivy 2.0+
- SQLite3
- ReportLab (للطباعة)
- FPDF2 (للطباعة على Android)
- Cryptography (للترخيص)
- PIL / Pillow (للصور)

### لبناء APK (Android)
- Buildozer
- Android SDK & NDK
- JDK 8+
